Base64: DECODE

Base64 é um grupo de esquemas de codificação binários para texto semelhantes que representam dados binários em um formato de string ASCII, traduzindo-os em uma representação radix-64.
O termo Base64 se origina de uma codificação de transferência de conteúdo MIME.

A codificação Base64 é frequentemente utilizada quando existe uma necessidade de transferência e armazenamento de dados binários para um dispositivo designado para trabalhar com dados textuais.

Esta codificação é amplamente utilizada por aplicações em conjunto com a linguagem de marcação XML, possibilitando o armazenamento de dados binários em forma de texto.
